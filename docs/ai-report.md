# Raport AI pentru proiectul TSS 2026

## Tool AI folosit
- GitHub Copilot / OpenAI ChatGPT pentru sugerarea testelor unitare si structura proiectului.

## Prompt folosit
```
Generate JUnit 5 tests for a TriangleClassifier class in Java that can classify triangles as invalid, equilateral, isosceles, scalene and right scalene. Include boundary and equivalence class cases.
```

## Teste autogenerate vs teste manuale
- `TriangleClassifierTest.java`: suite manuala concentrata pe strategii de testare si cazuri critice.
- `TriangleClassifierAIGeneratedTest.java`: teste mai simple, orientate pe verificari functionale directe.

### Tabel comparativ
| Tip test | Scop | Avantaj | Limitare |
|---|---|---|---|
| Manual | Partitionare, frontiera, mutanti | Acoperire mai detaliata, argumentat | Consum mai mare de timp |
| AI-generated | Prototip rapid | Produs automat rapid | Poate repeta cazuri sau lipsi valori de frontiera |

## Rezultate obtinute cu ajutorul AI

### Teste generate
- **Numar teste AI**: 5 teste functionale
- **Timp generare**: ~2 minute cu GitHub Copilot
- **Calitate**: Teste valide, dar incomplete pentru strategii avansate

### Comparatie rezultate testare
| Metric | Teste manuale | Teste AI | Interpretare |
|---|---|---|---|
| **Numar teste** | 7 | 5 | Teste manuale mai cuprinzatoare |
| **Acoperire linii** | 87% | 87% | Acelasi nivel de acoperire |
| **Mutanti ucisi** | 18/25 (72%) | Include in total | Teste manuale contribuie la detectia mutantilor |
| **Strategii acoperite** | Echivalenta, frontiera, decizii | Functionale de baza | AI lipseste valori critice si mutanti |

### Analiza calitatii
Testele AI au generat cod valid si au trecut toate verificarile, dar au acoperit doar cazuri functionale simple. Testele manuale au inclus explicit:
- Valori de frontiera (0, -1, inegalitati triunghi)
- Clase de echivalenta pentru toate tipurile
- Conditii pentru detectia mutantilor de tip CONDITIONAL_BOUNDARY

**Concluzie**: AI-ul este excelent pentru prototipare rapida, dar necesita completare umana pentru testare completa si robusta.

## Exemple de cod autogenerat
```java
@Test
void autoGeneratedShouldRejectNegativeSides() {
    assertFalse(TriangleClassifier.isValidTriangle(-3, 4, 5));
    assertEquals(TriangleType.INVALID, TriangleClassifier.classify(-3, 4, 5));
}
```
