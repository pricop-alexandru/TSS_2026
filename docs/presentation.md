# Tema proiectului

- Testare unitara in Java
- Clasificare triunghiuri
- Strategii: echivalenta, valori de frontiera, acoperire, mutanti

---

# Arhitectura solutiei

- `TriangleClassifier` clasa principala
- `TriangleType` enumerare
- `TriangleClassifierTest` teste manuale
- `TriangleClassifierAIGeneratedTest` teste autogenerate AI

---

# Strategii de testare

- Clase de echivalenta
- Analiza valorilor de frontiera
- Acoperire decizionala si conditionala
- Mutant testing cu PIT

---

# Instrumente

- Java 17
- Maven
- JUnit 5
- JaCoCo
- PIT
- GitHub Copilot / ChatGPT

# Concluzii

- Testele manuale asigura acoperire mai bogata
- Testele AI accelereaza prototiparea
- Analiza mutantilor creste increderea in suite
